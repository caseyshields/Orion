Detailed Description

The Tracker module represents a sensor on the surface of the earth at a specific time, observing a celestial object. It can transform star coordinates between celestial and horizon, using Novas 3.1 to handle orbit, spin, precession, parallax, atmospheric refraction and relativistic aberration, whatever that is.
