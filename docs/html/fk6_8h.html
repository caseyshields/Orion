int fk6_get_value ( char *  record,
FK6_Field *  field,
void *  dest 
)

Uses the given metadata to parse the field from the record and convert it to the appropriate type. Fields with a format of 'A' are converted to C strings. 'I' fields are converted to long integers. 'F' fields are converted to doubles. Make sure the destination pinter is appropriately typed!

Parameters
recordA line of text from a Vizier/CDS record
fieldthe metadata of the desired field
desta ponter whose value will be set to the value of the requested field.
