◆ crc

unsigned short crc

A 16-bit error checksum, Cyclic Redundancy Check. The CRC field consists of the Consultative Committee International for Telegraphy and Telephone (CCITT)/(ITU) V.41 polynomial, x^16 + x^12 + x^5 + 1, with the CRC accumulator preset ti 1's.

◆ sensor_type

unsigned short sensor_type

TIDC.

  • 0 Hex. Indicates IFF codes.
  • 1 Hex. Indicates RIU Sensor ID.
  • 2 Hex. Indicates GPS pod numbers.

◆ symbol_type

unsigned short int symbol_type

If source is an IFF system then byte 19 is zero filled. Byte 18's value is the Symbol type as follows:

  • 0 = Foe
  • 1 = Non-Player
  • 2 = Friend As enumerated in TATS_IFF_ETYPE

◆ tcn_time

unsigned short tcn_time

Unsigned integer. LSB = 1 millisecond. Maximum count is 59.999 seconds. TCN time is used to calculate latency. It is also used by some receiving systems for delta calculations.

◆ track_id

unsigned short int track_id

For single target sensors this field is zero filled. If source is a Multi-Target sensor then bytes 18 and 19 are the track ID.

◆ track_status

unsigned char track_status

A bitmask showing system status. Bit values are obtained by bitwise ORing values from TATS_STATUS_ETYPE
