Detailed Description

A summary of the IERS Earth Orientation Parameters. These parameters are produced by combining measurements from Very Long Baseline Interferometry (VLBI), Satellite Laser Ranging (SLR), the Global Positioning System (GPS) satellites, Lunar Laser Ranging (LLR), and meteorological predictions of variations in Atmospheric Angular Momentum (AAM).
